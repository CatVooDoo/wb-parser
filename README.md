# WB Parser — Парсер карточки товара Wildberries

Веб-приложение на чистом PHP 8.4 для парсинга карточек товаров Wildberries по артикулу (nmID).

## Возможности

- ✅ Поиск товара по артикулу (5–12 цифр)
- ✅ Получение данных: название, бренд, категория, описание, характеристики
- ✅ Галерея изображений с миниатюрами
- ✅ Раскрываемое описание товара
- ✅ Ссылка на товар на Wildberries
- ✅ Тёмный современный дизайн в стиле WB
- ✅ Адаптивность для мобильных устройств
- ✅ Обработка ошибок (невалидный артикул, товар не найден, таймауты)

## Требования

- PHP 8.4+
- Composer
- Guzzle HTTP Client (устанавливается автоматически)

## Установка

```bash
# Установите зависимости
composer install
```

## Запуск

```bash
# Встроенный сервер PHP
php -S localhost:8080
```

Откройте в браузере: **http://localhost:8080**

## Как использовать

1. Введите артикул товара Wildberries (например: `154109138`)
2. Нажмите **«Найти товар»** или Enter
3. Получите карточку товара с фото, описанием и характеристиками

## API Wildberries

Приложение использует публичный CDN Wildberries (`wbbasket.ru`) без токена.

### Алгоритм работы

1. **Вычисление VOL и PART:**
   - `VOL = (int)(nmID / 100000)`
   - `PART = (int)(nmID / 1000)`

2. **Определение сервера по таблице VOL:**
   - VOL ≤ 143 → basket-01
   - VOL ≤ 287 → basket-02
   - ... (см. код в классе `WbParser`)

3. **Формирование URL:**
   ```
   https://basket-{XX}.wbbasket.ru/vol{VOL}/part{PART}/{nmID}/info/ru/card.json
   ```

## AJAX API

Приложение поддерживает AJAX-запросы для интеграции.

### Запрос

```
GET /index.php?ajax=1&nm=154109138
```

### Ответ (успех)

```json
{
  "success": true,
  "data": {
    "nm_id": 154109138,
    "name": "Секатор профессиональный титановый ЦИ",
    "brand": "ЦИ",
    "category": "Секаторы",
    "description": "...",
    "options": [
      {"name": "Цвет", "value": "Серый"},
      {"name": "Материал", "value": "Титан"}
    ],
    "images": [
      "https://basket-10.wbbasket.ru/.../1.webp",
      "https://basket-10.wbbasket.ru/.../2.webp"
    ],
    "wb_url": "https://www.wildberries.ru/catalog/154109138/detail.aspx"
  }
}
```

### Ответ (ошибка)

```json
{
  "success": false,
  "error": "Товар не найден (артикул: 999999999999)"
}
```

### Типы ошибок

| Ситуация | Сообщение |
|----------|-----------|
| Некорректный артикул | `Некорректный артикул` |
| Товар не найден | `Товар не найден (артикул: XXXX)` |
| Сервер недоступен | `Сервер WB не отвечает, попробуйте позже` |
| Ошибка парсинга JSON | `Не удалось разобрать ответ WB` |

## Технологии

- **Бэкенд:** PHP 8.4 (строгая типизация)
- **HTTP-клиент:** Guzzle 7.10
- **Фронтенд:** HTML5 + CSS3 + Vanilla JS
- **Шрифты:** Google Fonts (Unbounded, Inter)
- **Стиль:** Тёмный минимализм, без фреймворков

## Особенности

- Один файл `index.php` — и бэкенд, и фронтенд
- Никаких фреймворков и CSS-библиотек
- CORS не требуется (всё на одном сервере)
- Изображения с CDN WB отображаются напрямую
